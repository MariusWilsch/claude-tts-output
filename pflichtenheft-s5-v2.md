# Section 5: Abnahmekriterien

## FR-01 Navigation Fragen (Beispiele aus Excel)

Test 1: Benutzer fragt "Wo finde ich die Texte für die Webseitendatenschutzerklärung?"

Erwartetes Ergebnis: Antwort referenziert Kapitel 8.

Test 2: Benutzer fragt "Wo finde ich eine Vorlage für eine Datenschutzverpflichtung?"

Erwartetes Ergebnis: Antwort referenziert Kapitel 09.

Test 3: Benutzer fragt "Wo finde ich Mustervorlagen für AV-Verträge?"

Erwartetes Ergebnis: Antwort referenziert Kapitel 05.

## FR-02 Streaming

Test: Anfrage wird gesendet.

Erwartetes Ergebnis: Erste Tokens erscheinen innerhalb von 4 Sekunden.

## FR-03 Follow-up

Test: Nach Frage "Wo finde ich Mustervorlagen für AV-Verträge?" fragt Benutzer "Was muss da rein?"

Erwartetes Ergebnis: System antwortet mit Bezug zu AV-Verträgen aus Kapitel 05.

## FR-04 OpenWebUI

Test: System ist deployed und Benutzer öffnet die URL.

Erwartetes Ergebnis: Chat-Interface ist zugänglich.

## FR-05 bis FR-07 Materialien

Test: Frage zu einem Dokument aus den ZIP-Vorlagen.

Erwartetes Ergebnis: Relevantes Dokument wird gefunden und referenziert.

## FR-08 und FR-09 Output

Test: Beliebige Anfrage wird gestellt.

Erwartetes Ergebnis: Antwort enthält nur Text, keine URLs oder Bilder.

## FR-10 Evaluation

Test: Alle Fragen aus Testfragen.xlsx werden gestellt.

Erwartetes Ergebnis: Über 90 Prozent werden korrekt beantwortet.
