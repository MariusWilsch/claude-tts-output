# Section 5: Abnahmekriterien

## FR-01 Fragen

Test: Benutzer fragt "Wo finde ich die TOM-Checkliste?"

Erwartetes Ergebnis: Antwort referenziert Kapitel 06.

## FR-02 Streaming

Test: Anfrage wird gesendet.

Erwartetes Ergebnis: Erste Tokens erscheinen innerhalb von 4 Sekunden.

## FR-03 Follow-up

Test: Nach vorheriger Frage zu Kapitel 06 fragt Benutzer "Was steht da drin?"

Erwartetes Ergebnis: System antwortet mit Bezug zum vorherigen Kontext.

## FR-04 OpenWebUI

Test: System ist deployed und Benutzer öffnet die URL.

Erwartetes Ergebnis: Chat-Interface ist zugänglich.

## FR-05 bis FR-07 Materialien

Test: Frage zu einem Dokument aus den ZIP-Vorlagen.

Erwartetes Ergebnis: Relevantes Dokument wird gefunden und referenziert.

## FR-08 und FR-09 Output

Test: Beliebige Anfrage wird gestellt.

Erwartetes Ergebnis: Antwort enthält nur Text, keine URLs oder Bilder.

## FR-10 Evaluation

Test: Alle Fragen aus Testfragen.xlsx werden gestellt.

Erwartetes Ergebnis: Über 90 Prozent werden korrekt beantwortet.
