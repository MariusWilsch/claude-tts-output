# Claude TTS Output

Temporary markdown hosting for text-to-speech integration with Speechify browser extension.

## Purpose

This repository hosts markdown content generated by Claude Code that is read aloud using the Speechify browser extension via GitHub Pages.

## Usage

- Content is stored in `current.md`
- Accessible via: https://mariuswilsch.github.io/claude-tts-output/current.md
- Open the URL in your browser with Speechify extension enabled

## Technical Details

- Repository is public (required for GitHub Pages on free account)
- Content in `current.md` is overwritten with each update
- GitHub Pages serves the latest pushed content
